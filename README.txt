LITTLE LEARNERS - install as an app (5 minutes, free)

1. Create a free account at github.com
2. New repository -> name it "little-learners" -> Public
3. "Uploading an existing file" -> drag ALL files from this folder -> Commit
4. Settings -> Pages -> Branch: main -> Save
5. Open https://YOURNAME.github.io/little-learners/ on the phone
6. Chrome menu -> "Add to Home screen" / "Install app"

Camera & microphone will now ask permission and work.
Progress (stars, stickers, treasures, levels) saves on the device.
To ship updates later: replace index.html AND bump VERSION in sw.js.
